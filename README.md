# ESP32 Fan Controller with Flutter App 

Using ESP32 microcontroller to control a custom fan (DC motor) with 2 operating modes: PWM mode and Automation
## Features
- PWM Mode: control fan speed via pwm signal, 4 signal: off , 1 , 2 , 3
- Automation mode: fan speed automatically controlled by temperature sensor
- Control via Bluetooth Classic BR/EDR
- Temperature and Humidity sensor
- Custom Android app showing status, Bluetooth connectivity, temperature and humidity
